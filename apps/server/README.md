# Server Application

The Node.js server providing API endpoints for the autofix platform.

## Features

- REST API for text completion
- WebSocket support for real-time updates
- Rate limiting
- Session management
- File storage integration

## Development

```bash
npm run dev
```

## API Endpoints

- `POST /api/complete` - Generate text completions
- `GET /api/status` - Check server status

## Technologies

- Express.js
- TypeScript
- WebSocket
- Session management
- Rate limiting

# Client Application

The web-based text editor interface for the autofix platform.

## Features

- Real-time text completion
- Multiple AI providers (Gemini, SambaNova)
- Customizable writing styles and modes
- Keyboard shortcuts
- Settings panel

## Development

```bash
cd apps/client
npm run dev
```

## Technologies

- React
- TypeScript
- Tailwind CSS
- Vite
- React Query
